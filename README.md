# IndicaMulheres
Repositório do projeto PyLadies Indicações, contendo documentação técnica, protótipos de interface e planejamento do sistema multiplataforma para cadastro, consulta e recomendação de profissionais e empreendimentos de mulheres. Inclui README detalhado, arquitetura proposta, lista de tecnologias e cronograma de desenvolvimento para a Etapa 2 (N708).
