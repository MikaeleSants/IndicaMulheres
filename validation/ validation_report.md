# Relatório de Validação – PyLadies Indicações

## 1. Apresentação do Projeto
- **Data da apresentação: 20/10/2025
- **Formato:** Apresentação online para representante da comunidade PyLadies Fortaleza
- **Conteúdo apresentado:**
    - Objetivo do projeto
    - Funcionalidades principais (cadastro, busca, consulta)
    - Arquitetura geral (frontend estático + backend reativo + MongoDB)

---

## 2. Feedback Recebido
- As participantes consideraram o projeto **bem estruturado e útil** para a comunidade.
- Foi solicitado **remover a integração com WhatsApp**, pois não existe um número oficial da comunidade para esse fim.
- As demais funcionalidades foram validadas como adequadas.
- Houve interesse em **adicionar ao site oficial da comunidade** quando a plataforma (o site delas) estiver pronta, para ampliar a divulgação.

---

## 3. Ajustes Implementados
- **Removida a integração com WhatsApp** da documentação e arquitetura do sistema.
- Mantida a integração com **EmailService (API Google)** para envio de notificações.
- Ajustes no UX da página, de acordo com o que foi pedido pela UX da comunidade
